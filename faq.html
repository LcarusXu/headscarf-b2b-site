Help & Fabric Guide

Questions, answered softly.

Use this as your first FAQ page. You can add shipping, return and sizing rules once your store policy is finalized.

What fabrics does NUVAYLE focus on?

Modal, chiffon, voile, jersey, cotton linen, printed hijab fabrics, fringe styles, instant hijabs, accessories and soft shawls.

Can I use this website package on GitHub Pages?

Yes. Upload the folder contents to a GitHub repository and enable GitHub Pages from the repository settings.

Are the images copied from the reference website?

No. All image files in this package are original SVG illustrations made for the NUVAYLE demo site.

Can I replace the product images later?

Yes. Keep the same file names for quick replacement, or update the image paths in data/products.json.

Is this site ready for Shopify checkout?

This package is static for GitHub Pages. It can be adapted to Shopify, WooCommerce or a custom backend later.

Does the site include wholesale pages?

Yes. The Wholesale and Contact pages are designed to collect B2B inquiries and custom order requests.
